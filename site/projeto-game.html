(Nome do Game)

O jogo

Se resume a uma realidade onde existem infinitos mundos onde é possível ser quem quiser e viver numa eterna exploração dos mais diversos lugares e siruações possíveis.

Funções

Moedas virtuais, interações com outros players, metaverso.

Regras

Existe apenas uma única regra no jogo que é o tempo diário permitido para jogar. Ainda vai ser criado o mecanismo para estabelecer como será esse controle.

Cronograma do Projeto

  1. Site e informações do projeto.
  2. Pesquisas e fundamentos dos primeiros passos.
  3. Verificar viabilidade.
  4. Iniciar criação do game.
  5. Testes de jogabilidade.
  6. Buscar investimentos para lancamento do jogo.
  7. Lançamento do game.
